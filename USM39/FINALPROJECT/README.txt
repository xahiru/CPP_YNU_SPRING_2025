University Student Management System (C++ Final Project)
=======================================================

## Compilation (Linux/macOS)
$ g++ *.cpp -o usms

## Compilation (Windows, using MinGW)
> g++ *.cpp -o usms.exe

## Running
$ ./usms   (Linux/macOS)
or
> usms.exe (Windows)

## Features
- Add/view students, faculty, courses
- Student-course enrollment
- Grade assignment and GPA calculation
- Robust input validation and exception handling
- Object-oriented and STL-based design

## Sample Data
- Run the program and use menu options to create test data.
